# Q11 - IoT Architecture

## Perception Layer
Collects data from the environment using sensors.

**Example:** Temperature Sensor (LM35)

## Processing Layer
Processes the data received from sensors.

**Example:** Arduino UNO

## Network Layer
Transfers data between devices using communication technologies.

**Example:** Wi-Fi

## Application Layer
Displays information to the user and allows interaction.

**Example:** Mobile Application
