# Q12 - Microcontroller vs Microprocessor

| Parameter | Microcontroller (Arduino UNO) | Microprocessor (Raspberry Pi) |
|------------|-------------------------------|-------------------------------|
| CPU Speed | 16 MHz | 1.5 GHz |
| On-chip RAM | 2 KB SRAM | 1 GB – 8 GB RAM |
| On-chip Flash | 32 KB Flash Memory | No on-chip flash, uses SD Card storage |
| Operating System | No Operating System required | Linux Operating System |
| Typical Use Case | Embedded systems, home automation, robotics | Computers, IoT servers, multimedia applications |
| Power Consumption | Low | Higher than a microcontroller |
