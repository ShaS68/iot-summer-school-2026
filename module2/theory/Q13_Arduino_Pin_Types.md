# Q13 - Arduino Pin Types

## 1. Digital Input

Digital input pins are used to read two states: HIGH (1) or LOW (0). They are commonly connected to switches and push buttons.

**Example:** Push Button

**IoT Use Case:** Detecting whether a door is open or closed in a smart home security system.

---

## 2. Digital Output

Digital output pins are used to send HIGH or LOW signals to control external devices.

**Example:** LED

**IoT Use Case:** Turning ON an LED to indicate that a motion sensor has detected movement.

---

## 3. Analog Input

Analog input pins read varying voltage values between 0V and 5V and convert them into digital values.

**Example:** Potentiometer

**IoT Use Case:** Reading temperature from an LM35 sensor for weather monitoring.

---

## 4. PWM Output

PWM (Pulse Width Modulation) pins generate variable output by changing the duty cycle. They are used to simulate analog output.

**Example:** LED Brightness Control

**IoT Use Case:** Controlling the speed of a DC motor in an automated irrigation system.

---

## 5. I2C / SPI Communication Pins

I2C and SPI are communication protocols used to exchange data between Arduino and external devices such as sensors and displays.

**Example:** OLED Display

**IoT Use Case:** Displaying temperature and humidity data collected from sensors on an OLED screen.
