# DC Motor Speed Controller

## Components Used
- Arduino Uno
- DC Motor
- TIP120 Transistor
- 1N4007 Diode
- Potentiometer
- Green LED
- Red LED
- 220Ω Resistors
- 1kΩ Resistor

## Description
This project controls the speed of a DC motor using PWM. A potentiometer is used to vary the motor speed. The PWM value is displayed on the Serial Monitor. A green LED indicates high speed (above 75%), and a red LED indicates low speed (below 25%).
