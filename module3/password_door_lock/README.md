# Password-Based Door Lock System

## Components Used
- Arduino Uno
- 4x4 Keypad
- 16x2 I2C LCD
- SG90 Servo Motor

## Description
This project implements a password-based door lock system. A user enters a password using a 4x4 keypad. If the password is correct, the LCD displays "Access Granted" and the servo motor rotates to open the door. After 5 seconds, the servo returns to its original position. If the password is incorrect, the LCD displays "Access Denied".
