# Servo Motor Position Controller

## Components Used
- Arduino Uno
- SG90 Servo Motor
- Potentiometer
- LED
- 220Ω Resistor

## Description
This project controls the position of a servo motor using a potentiometer. The potentiometer value is mapped to an angle between 0° and 180°. The current angle is displayed on the Serial Monitor. A warning LED turns ON whenever the servo angle exceeds 150°.
